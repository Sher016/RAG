# Frontend - RAG Chat

Interfaz en React + Vite + TypeScript para el proyecto RAG.  
Incluye un chat dinámico que cambiará de estilo según el género del texto (terror, fantasía, sci-fi, etc.).

## 🚀 Tecnologías
- React + Vite
- TypeScript
- TailwindCSS
- Axios

## ⚙️ Instalación
```bash
cd frontend
npm install
npm run dev
